# Prompt para Desenvolvimento Completo de Sistema de Gestão para Restaurantes e Bares

## Contexto do Projeto

Desenvolva um **Sistema de Gestão Completo para Restaurantes e Bares**, uma plataforma web multi-tenant moderna que permite gerenciar múltiplos estabelecimentos gastronômicos de forma independente e segura. O sistema deve ser desenvolvido com **React/TypeScript** no frontend, **Supabase** como backend e **PostgreSQL** como banco de dados, seguindo arquitetura multi-tenant com Row Level Security (RLS).

## Stack Tecnológica Obrigatória

### Frontend
- **Framework**: React 18+ com TypeScript
- **Build Tool**: Vite para desenvolvimento rápido
- **Styling**: Tailwind CSS para design system consistente
- **State Management**: React Context API + React Query para cache
- **Routing**: React Router DOM v6
- **Forms**: React Hook Form com validação Zod
- **UI Components**: Headless UI ou Radix UI para acessibilidade
- **Icons**: Lucide React ou Heroicons
- **Charts**: Recharts para dashboards e relatórios

### Backend e Infraestrutura
- **BaaS**: Supabase (Auth, Database, Storage, Real-time)
- **Database**: PostgreSQL com extensões Supabase
- **Authentication**: Supabase Auth com RLS
- **File Storage**: Supabase Storage para imagens
- **Real-time**: Supabase Realtime para atualizações instantâneas

### Ferramentas de Desenvolvimento
- **Package Manager**: npm ou yarn
- **Linting**: ESLint + Prettier
- **Testing**: Vitest + React Testing Library
- **Type Safety**: TypeScript strict mode
- **Git Hooks**: Husky para pre-commit
- **Deployment**: Vercel ou Netlify
